<h2 id="estructura">Estructura</h2>
<p>La web muestra una página principal en la que se pueden apreciar 11 cards, una por cada ejercicio realizado</p>
<br>
<img src="./img-readme/Portada.png" alt="imagen portada" style="width:100%">
<br>
<p>Ejercicio 0: Crea una página web, en la que, al ingresar, se muestre el mensaje “Hola mundo” mediante un Alert</p>
<p>Ejercicio 1: Crea una página web, en la que, al ingresar, se muestre el mensaje “Hola mundo” tras pulsar un botón</p>
<p>Ejercicio 2: Pide mediante formulario, el radio de una circunferencia y muestra tres resultados: longitud, área del círculo y volumen esfera</p>
<p>Ejercicio 3: Pide mediante formulario, los 3 lados de un triángulo y muestra de que tipo es</p>
<p>Ejercicio 4: Pide mediante formulario, el DNI solo con sus números y muestra la letra que le corresponde</p>
<p>Ejercicio 5: Pide mediante formulario, una cadena de cáracteres y muestra si está en mayúsculas, minúsculas o ambas</p>
<p>Ejercicio 6: Pide mediante formulario, una palabra y muestra si es palíndroma o no/p>
<p>Ejercicio 7: Pide mediante formulario, dos números y calcula su división entera con el algoritmo de Euclides</p>
<p>Ejercicio 8: Pide mediante formulario, dos números y calcula su MCD por el algoritmo de Euclides</p>
<p>Ejercicio 9: Pide mediante formulario, un número del 1 al 12 y muestra el nombre del mes que corresponde</p>
<p>Ejercicio 10: Pide mediante formulario, un mes y muestra su traducción</p>
<p>Ejercicio 11: Con una calculadoras creada en JS, crea una versión "Dark" y otra "Light"</p>
